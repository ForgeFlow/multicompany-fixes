=============================
Multicompany Property Product
=============================

This module is part of a set of modules that allow to set property fields on
a given model for multiple companies simultaneously. Specifically this module:

* Adds a new page 'Multi company' in the product and product category that
  allows the user to maintain, for all companies, the company-specific fields
  of one product.
